# Hi, I'm Guilherme Gabriel 👋

## About Me

I'm a Computer Engineering student (previously Bacharelor in Science and Technology) based in Brazil, with a background in low-level programming and embedded systems.

My main interest lies in **building neural networks and integrating ML models into real systems and applications** — from data pipelines to deployed solutions. Alongside that, I also have a strong interest in **Data Science**, driven by the understanding that good models start with good data analysis.

- 🎓 Studying **Computer Engineering**
- 🧠 Focused on **neural networks**, model building, and ML-integrated systems
- 📊 Interested in **Data Science** as a foundation for applied ML
- ⚙️ Background in **Embedded Systems**, Digital Signal Processing, and low-level C
- 🌱 Believer that understanding the fundamentals first makes you a better engineer at every level
